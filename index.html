Credit Card Fraud Detection

End-to-end ML project for detecting fraudulent transactions.

This Bachelor's Thesis develops a bank fraud detection model using Artificial Intelligence, based on a cost-sensitive Random Forest focused on minimizing the real economic impact of prediction errors. Using the public Kaggle Credit Card Fraud Detection dataset, several models are trained and evaluated, explicitly addressing class imbalance and the effect of transaction amount on decision cost. The system applies probability calibration and threshold optimization to balance precision, recall and operational cost. In addition, an interactive application is implemented to simulate real-time fraud detection and illustrate how the model could be integrated into FinTech platforms. The results show that the cost-sensitive approach improves the detection of relevant fraud cases while reducing financial losses and limiting user friction.

Portfolio Rebalancer

Python tool for rebalancing investment portfolios to target allocations.

Loads current holdings and target weights, computes the exact buy/sell orders required to rebalance, and shows the resulting post-trade portfolio composition in terms of units, values and weights. This allows you to correct portfolio drift, systematically realign the portfolio with a target strategic asset allocation and keep risk in line with the investor’s profile. The tool is designed to support recurring rebalance cycles and what-if scenarios (changing target weights, adding new assets, modifying contributions), making it a lightweight rebalancing engine that could be integrated into fintech tooling for portfolio management, robo-advisory workflows or internal risk-control dashboards.
